Download & Install KooRender

Launch & Start Rendering

KooRender

  1. Launch your 3ds Max software (2020 or newer version)
  2. Find and open the KooRender plugin from the 3ds Max plugin menu
  3. Log in with your Coohom/Kujiale account credentials
  4. Start using real-time rendering and offline rendering features for free
