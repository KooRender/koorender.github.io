# 快速开始

## 安装

安装过程：

- 点击下载好的安装包开始安装
- 点击下一步直到安装完成

**注**：我们会安装到所有支持的3ds Max版本

## 运行

运行方式：

- 启动3ds Max
- 在主菜单里面找到KooRender菜单栏
- 在KooRender菜单栏里面打开工具栏，工具栏会出现在3ds Max界面中
- 在KooRender工具栏中，点击输入框右侧的登录按钮
- 中国大陆用户请使用酷家乐(<https://kujiale.com>)帐号登录，海外用户请使用Coohom（<https://coohom.com>）帐号登录（没有账号的请先到各自官网注册）
- 登陆成功后，可以点击工具栏的第一个按钮打开实时渲染窗口（由于首次打开窗口需要上传整个场景的数据到渲染服务器，时间可能较长）
- 在弹出的渲染窗口出现渲染图像后，可以按照日常工作方式，操作和修改3ds Max场景，实时窗口会即时更新渲染图
