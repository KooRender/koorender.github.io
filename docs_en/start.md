# Quick Start

## Installation

Installation process:

- Click downloaded installaton package to start installation
- Click next button until the installation completes

**Remarks**: We will install for all supported 3ds Max versions

## Running

First steps:

- Launch 3ds Max
- Find KooRender menu in main toolbar
- Open KooRender toolbar in the menu, the toolbar will appear in 3ds Max UI
- In KooRender toolbar, click the login button on the right of input control
- Chinese mainland users should use Kujiale (https://kujiale.com) account to login, other oversea users should use Coohom (https://coohom.com) account to login (please sign up first if you don't have an account)
- After login, click the first button in toolbar to open real-time render view (since we need to upload the entire scene to render server for the first launch, this may take long time)
- After image appearing in the render view, you can work as usual in 3ds Max, the render view will update instantly
